rneos

Description

In this package the XML-RPC based API of the Network-Enabled Optimization System (NEOS) has been implemented. For more information see NEOS and for the API in particular API. A description of XML-RPC can be found here. The package utilizes S4-classes and methods and its documentation can be accessed here. The current version is: 0.2-7 as of 26 July 2012. The package does depend on the CRAN packages XML and RCurl and and the Omegahat package XMLRPC.

Documentation

The current documentation can be downloaded as pdf-file here.

Download

The package is hosted on CRAN and development versions can be installed from R-Forge. The CRAN release of the package can also be downloaded from this site in its source or binary form.
